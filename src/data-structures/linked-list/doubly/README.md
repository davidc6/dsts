# Doubly linked list

## Time complexity

- Insertion
  - At beginning - O(1)
  - At end - O(1)
  - Anywhere else - O(n)
- Deletion
  - At beginning - O(1)
  - At end - O(1)
  - Anywhere else - O(n)
- Read / search
  - At beginning - O(1)
  - At end - O(1)
  - Anywhere else - O(n)

## Space compexity

0(n)
